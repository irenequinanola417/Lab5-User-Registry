Debugging Notes – Lab 6

Bug Encountered:
While testing the program, the application crashed when a user entered text instead of a number for the age input.

Diagnosis:
I discovered that the program tried to convert the age input into an integer using int(). When letters such as "abc" were entered, Python raised a ValueError which caused the program to stop running.

Debugging Method:
I used print statements and tested different inputs in the program using the Python interpreter to observe where the error occurred.

Solution:
I implemented a try/except block to catch the ValueError exception. When invalid input is entered, the program now displays an error message and asks the user to enter a valid number again.

Result:
The program now handles incorrect inputs safely and does not crash when users enter invalid data.
